# Append

`append()` thêm một phần tử vào cuối danh sách.

## Cú pháp

`list.append(item)`

* `item`: phần tử cần thêm vào danh sách.

## Ví dụ

```python
fruits = ["apple", "banana"]

fruits.append("orange")

print(fruits)  # ['apple', 'banana', 'orange']
```

# Pop

`pop()` xóa và trả về phần tử trong danh sách.

## Cú pháp

`list.pop(index=-1)`

* `index`: *(tùy chọn)* vị trí phần tử cần xóa.
  Mặc định `-1` (phần tử cuối).

## Ví dụ

```python
fruits = ["apple", "banana", "orange"]

item = fruits.pop()

print(item)    # orange
print(fruits)  # ['apple', 'banana']
```

```python
fruits.pop(0)  # xóa phần tử đầu
print(fruits)
```
