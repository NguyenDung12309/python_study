# Enumerate

`enumerate()` dùng để lặp qua iterable và lấy **index + giá trị**.

## Cú pháp

`enumerate(iterable, start=0)`

* `iterable`: dữ liệu có thể lặp
* `start`: *(tùy chọn)* index bắt đầu, mặc định `0`

## Ví dụ

```python
todos = ["learn python", "read book", "exercise"]

for index, todo in enumerate(todos):
    print(index, todo)
```

```python
for index, todo in enumerate(todos, start=1):
    print(index, todo)
```

# zip()

`zip()` dùng để ghép nhiều iterable lại với nhau theo từng phần tử cùng vị trí.

## Cú pháp

`zip(iterable1, iterable2, ...)`

## Ví dụ

```python
names = ["An", "Bình", "Chi"]
scores = [8, 9, 10]

result = zip(names, scores)

print(list(result))
# Kết quả: [('An', 8), ('Bình', 9), ('Chi', 10)]
```
