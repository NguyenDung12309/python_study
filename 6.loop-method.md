# Enumerate

`enumerate()` dùng để lặp qua một iterable và lấy **cả vị trí (index) và giá trị** của phần tử.

```python
todos = ["learn python", "read book", "exercise"]

for index, todo in enumerate(todos):
    print(index, todo)
```

Kết quả:

```
0 learn python
1 read book
2 exercise
```

Có thể thay đổi giá trị bắt đầu của index:

```python
for index, todo in enumerate(todos, start=1):
    print(index, todo)
```
