# Input

Lấy dữ liệu do người dùng nhập vào.

## Cú pháp

`input(prompt)`

* `prompt`: Chuỗi thông báo hiển thị trước khi người dùng nhập dữ liệu.

# Open

`open()` dùng để mở file để đọc hoặc ghi dữ liệu.

## Cú pháp

`open(file, mode)`

* `file`: đường dẫn tới file.
* `mode`: chế độ mở file.

## Mode phổ biến

* `"r"`: đọc file *(mặc định)*
* `"w"`: ghi file (xóa nội dung cũ)
* `"a"`: ghi thêm vào cuối file
* `"x"`: tạo file mới

## Ví dụ

```python"
file = open("todos.txt", "r")
content = file.read()

print(content)

file.close()
```

# All()

`all()` dùng để kiểm tra **tất cả phần tử trong một iterable** có giá trị True hay không.

## Cú pháp

`all(iterable)`

* `iterable`: đối tượng có thể lặp (list, tuple, set, dictionary, ...).

## Hoạt động

* Trả về `True` nếu **mọi phần tử** đều là True.
* Trả về `False` nếu **có ít nhất một phần tử** là False.
* Iterable rỗng → trả về `True`.

## Ví dụ

```python
# Kiểm tra tất cả phần tử > 0
numbers = [1, 2, 3, 4]
print(all(n > 0 for n in numbers))  # True

# Có phần tử False (0 được xem là False)
mixed_list = [1, 0, 3]
print(all(mixed_list))  # False
