# Input

Lấy dữ liệu do người dùng nhập vào.

## Cú pháp

`input(prompt)`

* `prompt`: Chuỗi thông báo hiển thị trước khi người dùng nhập dữ liệu.

---

# Open

`open()` dùng để mở file để đọc hoặc ghi dữ liệu.

## Cú pháp

`open(file, mode)`

* `file`: đường dẫn tới file.
* `mode`: chế độ mở file.

## Mode phổ biến

* `"r"`: đọc file *(mặc định)*
* `"w"`: ghi file (xóa nội dung cũ)
* `"a"`: ghi thêm vào cuối file
* `"x"`: tạo file mới

## Ví dụ

```python"
file = open("todos.txt", "r")
content = file.read()

print(content)

file.close()
```
