# Capitalize

`capitalize()` trả về một chuỗi trong đó **ký tự đầu tiên được viết hoa**, các ký tự còn lại viết thường.

```python
text = "hello WORLD"
print(text.capitalize())  # Hello world
```

---

# Title

`title()` trả về một chuỗi trong đó **chữ cái đầu tiên của mỗi từ được viết hoa**.

```python
text = "hello world from python"
print(text.title())  # Hello World From Python
```

# Strip

`strip()` loại bỏ mọi khoảng trắng ở đầu và cuối chuỗi.

```python
text = "   hello world   "

result = text.strip()

print(result)  # hello world
```
