# Variables

## Quy tắc đặt tên biến trong Python

* Phải bắt đầu bằng **chữ cái** hoặc **dấu gạch dưới (_)**
* Không được bắt đầu bằng **số**
* Chỉ được chứa **chữ cái, số và dấu gạch dưới** (`A–Z`, `a–z`, `0–9`, `_`)
* Tên biến **phân biệt chữ hoa và chữ thường**
  (`firstname`, `Firstname`, `FirstName`, `FIRSTNAME` là các biến khác nhau)

# __name__

`__name__` là **biến đặc biệt (dunder variable)** mà Python tự động tạo cho mỗi file `.py` (module).

Nó cho biết **file hiện tại đang được chạy trực tiếp hay đang bị import từ file khác**.

---

## Cách hoạt động

### 1. Khi chạy file trực tiếp

Giả sử có file `main.py`:

```python
print(__name__)

if __name__ == "__main__":
    print("Running as main program")
```
