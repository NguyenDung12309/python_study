# Writelines

`writelines()` dùng để ghi nhiều dòng vào file.

## Cú pháp

`file.writelines(iterable)`

* `iterable`: iterable chứa các chuỗi cần ghi.

## Ví dụ

```python
lines = ["Learn Python\n", "Read book\n", "Exercise\n"]

file = open("todos.txt", "w")
file.writelines(lines)
file.close()
```

# Read

`read()` dùng để đọc toàn bộ nội dung hoặc một phần nội dung từ file.

## Cú pháp

`file.read(size)`

* `size` (optional): số ký tự cần đọc. Nếu bỏ qua → đọc toàn bộ file.

## Ví dụ

```python
file = open("data.txt", "r")

content = file.read()
print(content)

file.close()
```

# Readline

`readline()` dùng để đọc một dòng từ file.

## Cú pháp

`file.readline()`

## Ví dụ

```python
file = open("data.txt", "r")

line1 = file.readline()
line2 = file.readline()

print(line1)
print(line2)

file.close()
```

---

# Write

`write()` dùng để ghi một chuỗi vào file.

## Cú pháp

`file.write(string)`

* `string`: nội dung cần ghi vào file.

## Ví dụ

```python
file = open("notes.txt", "w")

file.write("Learn Python\n")
file.write("Practice coding\n")

file.close()
```
